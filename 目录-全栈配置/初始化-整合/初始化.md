### 初始化

#### 下载

```
git clone https://github.com/6byte/init.git
```

#### 初始化项目

```
cnpm init -y
cnpm install webpack
cnpm install vue-cli
vue init webpack 项目名(不能大写或中文)
cnpm run dev
```



将VUE编译成本地文件

```
--	打开文件
config/index.js

--	修改
将	assetsPublicPath: '/';	
改成	assetsPublicPath: './';

--	打开另一个文件文件
build/webpack.base.confg.js

--	修改
将	publicPath: process.env.NODE_ENV === 'production' ? config.build.assetsPublicPath : config.dev.assetsPublicPath

改成	publicPath: './'
```

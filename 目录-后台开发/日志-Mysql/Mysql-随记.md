存在问题



| 问题 |                             现象                             |
| :--: | :----------------------------------------------------------: |
| 脏读 |          一个事务，读取到另一个事务中没有提交的数据          |
| 虚读 |              同一个事务，两次读取到的数据不一样              |
| 幻读 | 一个事务操作表，另一个事务添加数据，则第一个事务查询不到修改 |
|      |                                                              |



隔离级别

|       类型       |          问题          | 效率 |
| :--------------: | :--------------------: | :--: |
| read uncommitted | 脏读，不可重复度，幻读 | 最高 |
|  read committed  |    不可重复度，幻读    |  高  |
| repeatable read  |        可重复读        |  低  |
|   serializable   |     可解决所有问题     | 最低 |
|                  |                        |      |

## Vue-基本应用

#### 高级选项卡

概览

```
实现效果:点击按钮，显示对应组件，相当于选项卡，但按钮可以随意摆放，不必一一对应
```

代码

```vue
<template>
    <div>
        <span>IP管理</span>
        <template v-slot:actions>
        <v-btn text @click="view('ip')">IP管理</v-btn>
        <v-btn text @click="view('user')">用户管理</v-btn>
        <v-btn text @click="view('sql')">SQL监控</v-btn>
    </div>
    
    <div >
        <ip v-show="showList.ip"></ip>
        <sql v-show="showList.sql"></sql>
        <user  v-if='showList.user'></user>
    </div>
</template>
```

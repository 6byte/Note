## SYS-进程线程

```
PCB
	1.进程标识信息：用户表示，父进程
	2.处理机状态信息:保存进程的运行现场
	3.用户可见寄存器:用户可以使用的数据
	4.控制和状态寄存器:PC,PSW
	5.栈指针：过程调用/系统调用/中断处理都需要
生命周期
	创建，运行，等待，唤醒。结束
线程:进程中的一条执行流程，进程将执行状态交给线程管理。
```

## SYS-信号量

```
作用：解决互斥，条件同步。解决线程同步问题！
分类：
	二进制信号量:0或1
	计数信号量:非负数
使用：
    1.初始化，给一个非负数。
    2.仅能通过原子操作wait(S),signal(S)访问

注：原子操作，不可中断

死锁：相互持有锁
读者-写者问题
哲学家就餐问题
```

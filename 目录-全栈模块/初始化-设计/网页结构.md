### 登录页面

##### 模块名

````
Login
````



##### 数据库

| 表名  |           字段           |
| :---: | :----------------------: |
| users | .username .passwd .phone |

##### 行为

|     行为     |           属性           | 表名  | 接口         |
| :----------: | :----------------------: | ----- | ------------ |
|    登录()    |    .username .passwd     | users | /login/login |
|    注册()    | .username .passwd .phone | users | /login/logup |
| 第三方登录() |           ---            |       |              |



##### 交互

```
1.提供清除密码
2.记住密码
```

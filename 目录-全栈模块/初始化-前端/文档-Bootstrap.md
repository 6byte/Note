### BootStrap备忘

文本

```
lead:字体变宽
blockquote-footer:字体变浅,前面单线

横向字体
<ul class="list-inline">
  <li class="list-inline-item">Lorem ipsum</li>
  <li class="list-inline-item">Phasellus iaculis</li>
  <li class="list-inline-item">Nulla volutpat</li>
</ul>

text-wrap :文字换行
text-truncate：截断文字加省略号
text-reset:重置颜色
垂直对齐:
	1.align-baseline
	2.align-top
	3.align-middle
	4.align-bottom
	5.align-text-top
	6.align-text-bottom
```

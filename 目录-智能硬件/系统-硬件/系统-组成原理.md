## 硬件-显卡显示器

硬件-显卡

```mysql
#分类
显卡芯片分为三种:集成显卡，核心显卡，独立显卡

#品牌:
A卡，N卡。两个品牌均是芯片生产者

#版本:
公版显卡：用显卡芯片厂家提供的标准规格制定的显卡
非公版显卡：对显卡供电，核心频率强化或弱化的显卡，
		一线非公版显卡:华硕，技嘉，蓝宝石
		二线非公版显卡:索泰，讯景，影驰，七彩虹，映众，耕升
		三线非公版显卡:铭瑄，盈通，昂达
		杂牌：铭影，精影
		
N卡版本:GTX,RTX系列
A卡版本:rx,vega系列

#性能参数
1.显存位宽	有64位，128位，192位

2.显存带宽 
	1.显存与显卡芯片之间的的数据传输量，其大小由显存位宽决定
	2.计算公式：显存带宽=显存频率*显存位宽/8

3.显存频率 显存的数据传输速度

4.显存容量 显存可缓冲存储数据的容量。


```



硬件-显示器

```
频率:60hz,75hz,144hz,240hz
帧数:一张图片
```
